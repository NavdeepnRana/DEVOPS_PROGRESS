Other Important Commands:

docker exec -it <container_id> bash
Gives access to the container’s terminal for internal operations

docker logs <container_id>
Shows logs/output generated by the container

docker inspect <container_id>
Provides detailed information about the container in JSON format

Volume Commands:

docker volume create myvolume
Creates a volume to store data permanently

docker volume ls
Shows all available volumes

docker volume inspect myvolume
Displays detailed information about a specific volume

docker volume rm myvolume
Deletes the volume
